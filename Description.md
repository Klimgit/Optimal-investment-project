
-  Данные: Тор-3000 акций США, 1979-2024 (10 593 тикера, 11 346 торг. дней)
	- OOS: 2010-2024, 179 ребалансировок, ~2 500 акций в универсе
	- Библиотека: quant-pml (единый подход бэктестинга)
-  Трейн не должен видеть будущее
- Качественные фичи, учесть волатильность
- Бэйзлайн - ридж + л2, логрег для задач классификации. Относительно их уже смотерть метрики дл моделей
- Две модели для регресси: MLP с двумя скрытыми слоями и smooth l1 loss, LSTM
	- 12 месячных снимков признаков, пихаем в lstm и предсказываем будущую доходность
	- Для классификации с bace дропаут в качестве регуляризации
- Для каждого месяца берем последние 60 месяцев и для них учим модель
	- Использщуем retrain: обучение с нуля каждый месяц
	- нормализация только на трейне
- Архитектура бэктест-пайплайна:
	- Признаки/риск-признаков
		- 8 momentum
			-  $r_{3},r_{6}, r_{12}$ - кумулятивные доходности (3, 6, 12 месяцев)
			- $r_1$ - краткосрочная доходность
			- $r_{3}^n,r_{6}^n, r_{12}^n$ - нормированные на волатильность
			- $\sigma$ - годовая волатильнотсь ( $63\cdot\sqrt{ 252 }$)
		- 16 MACD /Signal-признаков
			- 8 пар EMA-окон: (8,24), (16,48), (32, 96), ...
			- $MACD = \dfrac{EMA_{fast} - EMA_{slow}}{price}$
			- $Signal = \dfrac{EMA_{9}(MACD)}{price}$
		- Momentum = сколько выросло, MACD - как именно росло
			- Два актива с одинаковой доходностью могут расти по-разному
		- Итого: 24 признака на актив $x\in\mathbb{R}^24 = \mathbb{R}^8(momentum\setminus risk) + \mathbb{R}^16(MACD\setminus Signal)$
		- Нормализация
			- Z-score, параметры $\mu, \sigma$ - только из train окна
	- Скоринг
		- $s_i = f_{\Theta}(x_i)$
		- Ранжирование всех активов
	- Портфель
		- Top/Bottom 10%
		- Long/Short
		- Equally-weighted
	- Хедж
		- SPX futures
		- $\beta\approx 0$
		- Доллар-нейтраль
- Не смотрим на те акции, которые появились недавно, мы не можем на них учиться и не смотрим на те акции, которые делистнуты
